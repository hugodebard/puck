# Select

Render a `select` input with a list of options. Supports optional search functionality for better UX with long lists. Extends [Base](base).

<ConfigPreview
  label="Basic Select"
  componentConfig={{
    fields: {
      textAlign: {
        type: "select",
        options: [
          { label: "Left", value: "left" },
          { label: "Right", value: "right" },
        ],
      },
    },
    render: ({ textAlign }) => {
      return <p style={{ textAlign }}>Hello, world</p>;
    },
  }}
/>

## Basic Usage

```tsx {5-11} copy
const config = {
  components: {
    Example: {
      fields: {
        textAlign: {
          type: "select",
          options: [
            { label: "Left", value: "left" },
            { label: "Right", value: "right" },
          ],
        },
      },
      render: ({ textAlign }) => {
        return <p style={{ textAlign }}>Hello, world</p>;
      },
    },
  },
};
```

## Params

| Param                                     | Example                                               | Type     | Status   |
| ----------------------------------------- | ----------------------------------------------------- | -------- | -------- |
| [`type`](#type)                           | `type: "select"`                                      | "select" | Required |
| [`options`](#options)                     | `options: [{ label: "Option 1", value: "option-1" }]` | Object[] | Required |
| [`placeholder`](#placeholder)             | `placeholder: "Select an option..."`                  | String   | Optional |
| [`searchable`](#searchable)               | `searchable: true`                                    | boolean  | Optional |
| [`searchPlaceholder`](#searchplaceholder) | `searchPlaceholder: "Search..."`                      | String   | Optional |

## Required params

### `type`

The type of the field. Must be `"select"` for Select fields.

```tsx {6} copy
const config = {
  components: {
    Example: {
      fields: {
        textAlign: {
          type: "select",
          options: [
            { label: "Left", value: "left" },
            { label: "Right", value: "right" },
          ],
        },
      },
      // ...
    },
  },
};
```

### `options`

The options for the select field. The `value` can be a String, Number, Boolean, or Object.

```tsx {7-10} copy
const config = {
  components: {
    Example: {
      fields: {
        textAlign: {
          type: "select",
          options: [
            { label: "Left", value: "left" },
            { label: "Right", value: "right" },
          ],
        },
      },
      // ...
    },
  },
};
```

## Optional params

### `placeholder`

The placeholder text for the select field.

```tsx {7} copy
const config = {
  components: {
    Example: {
      fields: {
        country: {
          type: "select",
          placeholder: "Select a country...",
          options: [
            { label: "France", value: "fr" },
            { label: "Germany", value: "de" },
            // ...
          ],
        },
      },
      // ...
    },
  },
};
```

### `searchable`

Enable search functionality. When `true`, a search input appears at the top of the dropdown.

```tsx {7} copy
const config = {
  components: {
    Example: {
      fields: {
        country: {
          type: "select",
          searchable: true,
          options: [
            { label: "France", value: "fr" },
            { label: "Germany", value: "de" },
            // ...
          ],
        },
      },
      // ...
    },
  },
};
```

### `searchPlaceholder`

The placeholder text for the search input when `searchable` is `true`.

```tsx {8} copy
const config = {
  components: {
    Example: {
      fields: {
        country: {
          type: "select",
          searchable: true,
          searchPlaceholder: "Search countries...",
          options: [
            { label: "France", value: "fr" },
            { label: "Germany", value: "de" },
            // ...
          ],
        },
      },
      // ...
    },
  },
};
```
